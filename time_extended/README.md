###A simple gem for extension the Ruby class `DateTime`

####Install:
`gem install time_extended`

####Usage:
* DateTime.now.ymd => strftime("%y %m %d")
* DateTime.now.Bd => strftime("%B %d")
* DateTime.now.M3N => strftime("%M %3N")
